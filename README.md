# ADITYA-L1-MISSION-DATA-ANALYSIS
This document presents a comprehensive analysis workflow for Aditya-L1 Level-2 magnetometer data, demonstrating the extraction, processing, and visualization of Interplanetary Magnetic Field (IMF) measurements from India&amp;#39;s first solar mission. The analysis focuses on data from February 27, 2026, captured at the Sun-Earth Lagrange point L1.
